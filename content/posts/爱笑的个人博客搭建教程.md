## 技术选型

本次搭建使用了以下技术栈：

| 技术 | 用途 | 版本 |
|------|------|------|
| Next.js | 前端框架 | 16.x |
| React | UI 库 | 19.x |
| Tailwind CSS | 样式方案 | v4 |
| Framer Motion | 动画效果 | 12.x |
| TypeScript | 类型安全 | 5.x |

## 项目初始化

使用 `create-next-app` 创建项目：

```bash
npx create-next-app@latest my-blog --typescript --tailwind --app
```

## 配置样式

Tailwind CSS v4 使用全新的配置方式，通过 CSS 导入：

```css
@import "tailwindcss";
```

## 编写页面组件

使用 Next.js App Router 创建页面：

```tsx
// app/page.tsx
export default function Home() {
  return (
    <main>
      <h1>Hello, Blog!</h1>
    </main>
  );
}
```

## 部署到 Vercel

1. 推送代码到 GitHub
2. 在 Vercel 中导入项目
3. 自动部署完成

## 总结

搭建博客的过程本身就是一种学习。享受这个过程吧！
